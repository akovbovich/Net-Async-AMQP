
ConnectionManager

->add

Adds a given host to the pool.

Example:

 $cm->add(
  host     => 'localhost',
  port     => 5672,
  user     => '...',
  password => '...',
  vhost    => 'xxx',
 )

->request_channel

Assigns a spare channel.

Example:

 $cm->request_channel(
  qos => 10,
 )

->release_channel

Releases the given channel again.

Example:

 $cm->release_channel($ch)
